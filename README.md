# 🎮 GAME VAULT

Sistema de catálogo de jogos com interface visual estilo Steam.

## Funcionalidades
- CRUD completo (Cadastrar, Listar, Editar, Excluir)
- Interface gráfica com cards visuais e upload de capas
- Tema retro-gamer com dark mode e fonte pixel
- Banco de dados MySQL na nuvem (Aiven)
- Validação de dados e feedback visual pro usuário

## Tecnologias
- Java 21 + JavaFX
- MySQL 8 (Aiven Cloud)
- CSS customizado
- Maven
